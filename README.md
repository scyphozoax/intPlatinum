# intPlatinum

## 介绍
intPlatinum是使用`PyQt5`制作的开源即时聊天软件。支持发送文字和图片消息，并且支持图片在聊天窗口内显示。

## 特点
- 聊天内容（文字、图片、日志）不会永久储存在服务器上；

- 部署简单；

- 占用内存小。
